Pinging Application

Sockets
Servers
Python
API

Developed a Ping application in Python using ICMP. Ping application sends ping requests to a specified host. Each message contains a payload of data that includes a timestamp, ICMP header, checksum, sequence number, and time-to-live. After sending each packet, the application waits up to one second to receive a reply. If one second goes by without a reply from the server, then the client assumes that either the ping packet or the pong packet was lost in the network (or that the server is down). This program required use of RAW sockets
